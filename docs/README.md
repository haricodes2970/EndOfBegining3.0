# EndOfBeginning — Project Overview

Farewell website for the **BE AI & ML batch of 2022–26**, Jyothy Institute of Technology, Bengaluru.

Built and designed as a cinematic, emotional tribute — not a typical college page.

---

## Concept

> *"This isn't a goodbye. It's a graduation of the soul."*

The site is structured like a film — six chapters that tell the story of four years, the people in them, and where they go next.

---

## Tech Stack

| Layer | Choice | Reason |
|---|---|---|
| Framework | React 18 + TypeScript | Component reuse, type safety |
| Build tool | Vite | Fast HMR, minimal config |
| Routing | React Router DOM v6 | Clean multi-page SPA routing |
| Animations | GSAP + ScrollTrigger | Scroll-driven, high-quality animations |
| Styling | CSS Modules (per-page) + globals.css | Scoped styles, shared design tokens |
| Assets | Vite static imports | Logo, images handled natively |
